sqlmap的简单使用

安装sqlmap

sqlmap是基于Python环境的，所以现在电脑上安装Python，在py官网下载就好注意要下载2.7.15版的。（py官网是Python.org）

然后双击下载的python-2.7.15.amd64.msi文件进行安装

然后根据目录提示下一步，下一步安装就好，要记住py的安装路径，安装过后在环境变量配置。进入py的安装目录中，在地址栏上复制当前路径，然后右键我的电脑–>属性–>高级系统设置

然后选择环境变量

在系统变量中选择Path，双击进行编辑，在内容最后添加英文状态的分号，然后把py的路径复制到后面然后确定–>确定。

配置好环境变量后测试一下是否能运行py。win+r输入cmd，然后再命令提示符中输入python -V。如果能出现py版本则配置正确，如果不能请到网上自行百度Python环境变量的配置。

py环境配置完了后我们开始安装sqlmap，到GitHub上搜索sqlmap

然后下载

下载后把文件解压出来。进入解压出来的文件中，在路径中输入cmd或者shift+右键选择在此处打开命令窗口

输入python sqlmap.py

如果大家觉得这样打开sqlmap很麻烦可以进行如下操作。
在环境变量配置页中找到PATHEXT选项，双击进行编辑。在最后出输入“.py”,然后找到path把sqlmap的路径复制进去，加到系统变量中。

此方法是把.py的文件加到变量中，虽然有很多方便，同时别人干你的电脑也方便了很多，所以请慎重选择此方法。
下面赘述一下sqlmap的简单使用

sqlmap –url “域名” //这条命令是使用sqlmap查看目标网站的，反馈的内容里能看到很多的东西。可以看到测试网站使用的方法，还有网站是否在使用waf。

sqlmap -U “域名” –dbs //这条命令是使用sqlmap来跑目标网站的数据名字的

sqlmap -U “域名” -D “数据库名” –tables // 用来跑指定数据库中的表

sqlmap -U “域名” -D “数据库名” -T “表名” –columns // 用来跑指定表中的字段

qlmap -U “域名” -D “数据库名” -T “表名” -C “字段（可多选逗号分隔）” –dump //跑字段中的内容
